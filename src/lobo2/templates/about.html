About SUNET datasets

SUNET Datasets is an experimental service to provide a distribution infrastructure for large datasets using bittorrent technology. In order to use this service you must create a torrent metadata file (aka torrent file) and use a torrent software to provide access to the data. No data is stored or served from this service.

Creating a torrent (Linux/Unix)

A simple way to create a torrent metadata file is to use ctorrent: 

# ls example
data1.ex data2.ex data3.ex
# ctorrent -t -u {{ request.base_url }}/announce -c "Example dataset" -s example.torrent example
After this is done, you can upload example.torrent here and start serving your dataset by launching ctorrent: ctorrent -d -e 0 example.torrent. This will put ctorrent in daemon mode and cause it to seed indefinitely. For managing lots of datasets a better torrent client might be transmission, rtorrent or deluge.
